# Tally Prime Accounting System

## Project Overview

This project demonstrates a complete accounting system created using Tally Prime.

## Features

* Company Creation
* Chart of Accounts
* Stock Items and Stock Groups
* Purchase Vouchers
* Sales Vouchers
* Receipt Vouchers
* Payment Vouchers
* Stock Summary
* Outstanding Debtors
* Outstanding Creditors
* Profit and Loss Account
* Balance Sheet
